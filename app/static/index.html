常见问题

关于 XF模具智能体平台，你可能想知道的

如何开始使用XF模具智能体？

登录平台后即可直接使用XF模具研发智能体和XF模具质量智能体，无需任何配置。研发智能体协助模具设计、工艺规划与试模诊断；质量智能体提供VDA6.4体系审核、CAPA建议与过程分析。上传企业知识库文档后，智能体回答将更加精准。

Agent 的行为是否安全可控？

完全可控。XF模具智能体平台 内置安全护栏机制，你可以为 Agent 设定行为边界、输出规则与敏感词过滤。每次 Agent 的工具调用、推理路径与输出结果均有完整日志，支持全链路追溯审计。

支持哪些工具和集成？

内置知识库检索、联网搜索、数据分析、文档生成等核心工具。企业版支持通过 RESTful API 对接企业内部系统（如 CRM、ERP、OA），也可自定义 Python/Node.js 工具插件，实现深度集成。

Agent 能同时处理多个任务吗？

可以。XF模具智能体平台的 ReAct 推理引擎支持多步任务自动拆解与并行执行。Agent 会根据任务复杂度自主规划执行链路，串联多个工具调用，最终汇总结果交付。

免费版和企业版有什么区别？

免费版可创建 3 个 Agent，每日 20 次对话，适合个人体验评估。个人版支持 50 个 Agent 与每日 200 次对话，解锁联网搜索与深度思考。企业版不限 Agent 数量与对话次数，支持私有化部署、API 集成与专属技术支持。
